This section demonstrates the biblatex package with author-year citations.

\subsection{Basic Citations}

Here are some example citations using biblatex:

\begin{itemize}
    \item Parenthetical citation: \parencite{knuth1986texbook}
    \item Text citation: \textcite{lamport1994latex}
    \item Multiple citations: \parencite{knuth1986texbook, lamport1994latex}
\end{itemize}

\subsection{Citation Not in Database}
